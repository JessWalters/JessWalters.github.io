MY PROJECTS

Here are some of my most recent projects.


Vinny

Vinny is a bot made to moderate and interact with Discord Servers. Vinny now has over 165,000 users across 7,000 servers. Written in Java and Python.

Senior Design

My senior design project is focused around working with Danfoss to take in and analyse data from the MES system on their assembly lines. The project involves a lot of DB work and big data analytics.

Gotta-Go-Fast is a project I did solo for the fall 2017 HackISU hackathon. The project is a self-learning autonymous driver for all driving video games. I utilized a convoluted nueral net to interpret the game screen like a normal player and react accordingly.

Jangle is a project I did with a team for Com S 309. Jangle is a Text/Voice chat app for desktop that allows users to communicate effectivly. I worked in JavaFX and Java on the front end of the desktop application.
